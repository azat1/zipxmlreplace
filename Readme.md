Утилита позволяет выполнить поиск замену в xml файлах которые находятся в ZIP

=================================================================================

Использование 

zipxmlreplace "шаблон файла zip" "что заменить" "на что"

к примеру

zipxmlreplace GKUOKS*.zip "иное сооружение" "иное сооружение (дорога)"
